Halide NaCl game of life

This may take a while to load (it downloads native client versions of glibc and pthreads if you don't have them already). You will probably also need to enable NaCl for non-chrome-web-store sources in about://flags and restart chrome.

This demo is running a Halide image processing routine compiled for x86-64 native client. The source code is here. It's heavily vectorized and parallelized, and on my machine is about 15 times faster than unoptimized C (5ms vs 80ms).
